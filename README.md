# go.dev-tutorial
Practice from https://go.dev/doc/tutorial/

## Getting started

```bash
cd hello 

go run .

go help
```

update module
```bash
go mod tidy

```

install
```bash
go env -w GOBIN=$GOPATH/bin

go install
```
